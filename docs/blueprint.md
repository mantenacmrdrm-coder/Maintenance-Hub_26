# **App Name**: Maintenance Hub

## Core Features:

- Equipment Database: Maintain a searchable database of all equipment, with key details such as make, model, purchase date, and maintenance history.
- Operation Management: Schedule and track maintenance operations for each piece of equipment, including setting dates and intervals.
- Global Stats Display: Shows metrics, like the number of total operations, realized operations and operations success rate
- Equipment Statistics: Generate detailed statistics for each piece of equipment, including total operations, completed operations, overdue operations, and success rate.  The system includes precalculated statistics.
- Preventative Alerting: Provide smart alerting for equipment needing preventative maintenance using machine reasoning tool
- Excel Export: Planning data can be exported for import into other tools
- Interactive charts and diagrams: Operators will have the capability to visualise their operations and statistics in clear formats.

## Style Guidelines:

- Primary color: Deep teal (#329999), evokes reliability.
- Background color: Light teal (#E0F8F7), maintains visual consistency but is muted.
- Accent color: Gold (#B8860B), draws attention and indicates important elements.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern, neutral feel.
- Use clear, geometric icons that are relevant to maintenance and equipment management.
- A clean, tab-based layout, separating the app into Equipment, Operations, Planning and Alerts sections.
- Subtle transitions when navigating between sections.
